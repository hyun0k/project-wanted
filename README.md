### Get started

* Clone the project

```bash
$ git clone https://github.com/hyun0k/project-wanted
$ cd flask-sqlalchemy-skeleton
```

* Install dependencies

```bash
$ virtualenv venv -p python3
$ source venv/bin/activate
(venv) $ pip install -r requirements.txt
```


* Run the server

```bash
(venv) $ python run.py
```
